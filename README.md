# The Little Book of Revelation 10

The Book folder contains the original English version of the Little Book in different file formats. Please, feel free to share these files!

The language folder contains the language files used on [The Little Book](https://eyesupandopen.org/) Website (The Little Book of Revelation 10). These are AI translations, so expect some mistranslations and always compare them with the original.

If you want to help translate The Little Book to your language, you can download the language files here and modify them with any text editor (e.g. [VSCodium](https://vscodium.com/)). After modification, you can then send the finished files via email. My email can be found on the about page of the Eyesupandopen.org website.
