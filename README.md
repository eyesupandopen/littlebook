# Languages of The Little Book

These are language files, which are used in [The Little Book](https://eyesupandopen.org/) Website (The Little Book of Revelation 10).

If you want to help translating The Little Book to your language, you can download the language the files here and modify them with any text editor. After modification you can then send the finished files via email. My email can be found in the about page of the Eyesupandopen.org website.
